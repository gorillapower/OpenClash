# Project Rhythm

## Sources Of Truth

- `<framework-root>/`: focused execution model for active delivery
- `<framework-root>/decision/`: durable product and contract truth
- `<framework-root>/architecture/`: intended implementation structure
- `<framework-root>/execution-plan.md`: current sequencing view
- tracker work items: operational execution queue

## Refresh Triggers

Always refresh when:
- a durable rule changes
- a structural boundary changes
- the next work items would be written differently because of new learning
- optional planning parents need to be introduced or revised

## Practical Rule

If a future work item would likely be written differently because of what was
just learned, update the plan and docs now.
