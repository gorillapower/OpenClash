# Clash Nivo — Packages

Pre-built packages for [luci-app-clashnivo](https://github.com/gorillapower/OpenClash).

## Install

```sh
# OpenWrt 22.03+ (ipk)
opkg install luci-app-clashnivo_*.ipk

# OpenWrt snapshot (apk)
apk add luci-app-clashnivo-*.apk
```
